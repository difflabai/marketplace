---
description: Manage projects and tasks in DreamSolve
---

Use DreamSolve to manage projects and tasks. You have access to these tools:

- **createProject**: Create a new project with a goal description
- **createTask**: Add a task to an existing project
- **createTasksFromPlan**: Parse a plan and create multiple tasks automatically
- **assignTask**: Assign a task to a team member

When creating projects, ask the user for a clear goal description.
When creating tasks from a plan, break the plan into actionable items.
